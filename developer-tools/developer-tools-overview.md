## Developer Tools

CMSMS module development is supported by diagnostic tools that scan your code for best practices, security issues, and structural compliance. Use these tools before submitting your module to the Forge or deploying to production.

### In This Chapter

- [cmscli](/docs/developer-tools/cmscli) — A command line interface for CMSMS automation. Modules can contribute their own CLI commands.
- [ModuleCheckerer](/docs/developer-tools/module-checkerer) — A CMSMS admin module that scans installed modules for compliance with best practices, security patterns, and structural requirements.
